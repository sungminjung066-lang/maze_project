# 🧩 VR Maze Game (Unity)

대학생 시절 제작한 **VR 기반 미로 게임 프로젝트**입니다.
Unity와 Oculus XR 환경을 활용하여 플레이어가 가상 공간에서 미로를 체험할 수 있도록 구현했습니다.

---

## 📦 실행 파일 (VR 전용)

본 프로젝트는 **VR 환경(Oculus 등)** 에서 실행되는 애플리케이션입니다.

APK 다운로드:
[maze.apk](Build/maze.apk)

※ VR 기기가 필요합니다.
※ VR을 제외한 타 기기에서는 정상 실행되지 않을 수 있습니다.

- **개발 기간**: 2024.03 ~ 2024.06
- **개발 인원**: 1인 프로젝트

## ▶️ 실행 방법

### 📱 APK 실행 (권장)

1. VR 기기(Oculus Quest 등)에 APK 설치
2. VR 기기에서 애플리케이션 실행

※ VR 환경이 필요합니다.
※ VR을 제외한 타 기기에서는 정상 실행되지 않을 수 있습니다.

---

### 🧑‍💻 Unity 프로젝트 실행 (개발 환경)

1. Unity Hub 실행
2. 프로젝트 폴더 열기 (`Assets`, `Packages`, `ProjectSettings` 포함된 경로)
3. `Assets/Scenes` 폴더 내 Scene 실행
4. Oculus / XR 장비 연결 후 Play

※ 최초 실행 시 Library 폴더가 자동 생성되며 시간이 다소 소요될 수 있습니다.
